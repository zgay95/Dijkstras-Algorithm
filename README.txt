Steps taken to execute ShortestPath.cpp

1. Ensure you have downloaded a g++ compiler for C++
2. Ensure the files 'ShortestPath.cpp' & 'GraphData.txt' are in the same directory.
3. Navigate to directory containing the files on the command line.
4. Run the command 'g++ -std=c++17 -Wall *.cpp' to compile the code.
5. Run the command 'a.exe' to run the program executable.
